delimited.path¶

This module defines Path objects. a Path object implements an interface though which a path to nested data can be defined using one or more path segments.

Path¶

class Path(value)¶

The abstract base class for Path objects. When subclassing Path the _encode and _decode methods must be overridden to handle an external representation. Path segments are stored internally in a list in the segments attribute.

_encode(value)¶

Encode list to external representation. This method must be overridden when subclassing.

_decode(value)¶

Decode a value in external representation format to a list. This method must be overridden when subclassing.
