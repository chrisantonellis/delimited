Terms¶

External Representation¶

The type and format of a hashable iterable used by the _encode and _decode methods of Path objects. This is specific to each Path subclass.

TuplePath

("key1", "key2", "key3")

DelimitedStrPath

"key1.key2.key3"

Tuple Path Notation¶

The external representation of TuplePath.

("key1", "key2", "key3")

Delimited String Path Notation¶

The external representation of DelimitedStrPath.

"key1.key2.key3"

Nested Data¶

Container type data that has other container types as values.

{
  "make": "Ford",
  "model": "Bronco",
  "exterior": {
    "color": "blue",
    "trim": {
      "level": "base"
    }
  }
}

Path¶

An group of path segments that represent a location within a nested data structure.

"key1.key2.key3"

Path Segment¶

A single piece of a path.

>>> from delimited.path import DelimitedStrPath
>>> d = DelimitedStrPath("key1.key2.key3")
>>> d[1]

"key2" # this is a path segment
